# Project notes for Claude

## Commit attribution

When creating git commits in this repo, the ONLY co-author trailer should be:

```
Co-Authored-By: titanx-ces <ankur.nairit@gmail.com>
```

DO NOT include the Claude/Anthropic co-author line. Just the titanx-ces line, nothing else. Example commit message:

```
feat: short description

Longer body explaining the change...

Co-Authored-By: titanx-ces <ankur.nairit@gmail.com>
```
